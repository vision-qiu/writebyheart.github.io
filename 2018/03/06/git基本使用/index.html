一、git工作流程

工作流程：

1. 克隆远程资源库中项目代码到本地（本地会生成.git隐藏文件和项目代码，项目代码为工作区）
2. 对工作区代码进行添加，修改，删除一系列编辑操作
3. add 代码到暂存区（index）
4. 如果远程资源库代码有其他人提交新的版本，则需要拉取并合并到本地代码
5. 查看合并代码是否有冲突，若有冲突，修改之后，再一次 add 到暂存区
6. commit 代码到本地仓库
7. push 代码到远程资源库
